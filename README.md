# Основы нейронных сетей: Лабораторная работа №1
## Основные задачи

### 1. Классификация геометрических фигур (TF & PyTorch)
Реализовали систему распознавания треугольников, кругов и квадратов.
* Провели перебор 18 конфигураций гиперпараметров (нейроны, функции активации, батчи).
* Сравнили эффективность библиотек: **PyTorch** показал точность **95.36%**, в то время как **TensorFlow** — **90.07%**.

### 2. Классификатор игры «Крестики-нолики»
Обучили модель классифицировать исходы партий на основе игровых полей 3x3.
* **Результат:** Точность **100%** на тестовой выборке.

### 3. Распознавание цифр MNIST
Классическая задача распознавания рукописных символов.
* Реализовали полную цепочку предобработки: инверсию, нормализацию и решейп.
* Провели успешное тестирование модели на собственном рукописном изображении (уверенность **80.6%**).
